1.  **Hover** over any node to see the **connection handles** (dots on the sides).
2.  **Click and Drag** a line from one handle to another node's handle.
3.  **Release** to create a connection arrow.
4.  Matches are flexible: Connect Code to Notes, AI Responses to Documents, etc.

## ğŸ¤– AI Power Features

The Canvas is AI-aware!

1.  **Select a Node**: Click any node to select it.
2.  **AI Actions Menu**: A floating menu will appear next to the selected node.
3.  **Generate Insights**: content-aware actions like "Explain This", "Generate Code", or "Summarize" will instantly create a NEW connected node with the AI's output.

## ğŸ“š Library Integration (Drag & Drop)

You can bring your uploaded files directly onto the canvas:

1.  Open the **Library** view.
2.  **Click and Drag** a document card from the list.
3.  **Drop** it anywhere on the Canvas.
4.  A **Document Node** creates automatically, linking to that file's content.
